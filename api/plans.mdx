## Overview

Display detailed information about CoinPaprika's free tier and paid plan options. Use this command to understand what's included in each tier and decide if you need to upgrade.

## Usage

```bash
coinpaprika-cli plans
```

## Options

<ParamField path="--output" type="string" default="table">
  Output format: `table` or `json`
</ParamField>

<ParamField path="--raw" type="boolean" default="false">
  JSON output without `_meta` wrapper
</ParamField>

## Upgrading

### Step 1: Get an API key

1. Visit [coinpaprika.com/api/pricing](https://coinpaprika.com/api/pricing)
2. Choose your plan (Starter, Business, or Enterprise)
3. Complete signup and payment
4. Receive your API key via email

### Step 2: Configure the CLI

```bash
# Option 1: Interactive setup
coinpaprika-cli onboard

# Option 2: Direct configuration
coinpaprika-cli config set-key <YOUR_API_KEY>
```

### Step 3: Verify

```bash
# Check key status
coinpaprika-cli key-info

# Test with a paid feature
coinpaprika-cli ticker-history btc-bitcoin --start 2024-01-01
```

Consider upgrading if you need:

  <Accordion title="Historical data">

  <Accordion title="Commercial use">

    Paid plans allow commercial use, redistribution, and embedding in products.
  </Accordion>
  
  <Accordion title="Advanced features">
    - Ticker history endpoint (Starter+)
    - ID mappings across platforms (Business+)
    - WebSocket streaming (Business+)
    - SLA guarantees (Enterprise)

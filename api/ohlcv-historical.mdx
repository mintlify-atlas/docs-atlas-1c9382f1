## Overview

Returns historical Open-High-Low-Close-Volume (OHLCV) candlestick data for a specific cryptocurrency. Essential for technical analysis and charting.

## Usage

```bash
coinpaprika-cli ohlcv <COIN_ID> --start <START_DATE> [OPTIONS]
```

## Parameters

<ParamField path="coin_id" type="string" required>
  Coin ID in format `symbol-name` (e.g., `btc-bitcoin`)
</ParamField>

<ParamField path="--start" type="string" required>
  Start date in ISO format (e.g., `2024-01-01`, `2024-01-01T00:00:00Z`)
</ParamField>

<ParamField path="--end" type="string">
  End date in ISO format (defaults to current date if not specified)
</ParamField>

<ParamField path="--interval" type="string" default="24h">
  Candle interval: `5m`, `15m`, `30m`, `1h`, `6h`, `12h`, `24h`
</ParamField>

<ParamField path="--limit" type="number" default="50">
  Maximum number of candles to return
</ParamField>

<ParamField path="--quote" type="string" default="usd">
  Quote currency (e.g., `usd`, `btc`, `eth`)
</ParamField>

<ParamField path="--output" type="string" default="table">
  Output format: `table` or `json`
</ParamField>

<ParamField path="--raw" type="boolean" default="false">
  Show raw JSON response without wrapper metadata
</ParamField>

## Examples

### Get Bitcoin daily candles for January 2024

```bash
coinpaprika-cli ohlcv btc-bitcoin --start 2024-01-01 --end 2024-01-31
```

### Get Ethereum hourly candles

```bash
coinpaprika-cli ohlcv eth-ethereum --start 2024-01-01 --interval 1h --limit 24
```

### Get Solana 15-minute candles

```bash
coinpaprika-cli ohlcv sol-solana --start 2024-03-01 --interval 15m --limit 100
```

### Get OHLCV data in JSON format

```bash
coinpaprika-cli ohlcv btc-bitcoin --start 2024-01-01 --output json
```

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `time_open` | string | Candle opening time (ISO 8601 format) |
| `time_close` | string | Candle closing time (ISO 8601 format) |
| `open` | number | Opening price |
| `high` | number | Highest price during period |
| `low` | number | Lowest price during period |
| `close` | number | Closing price |
| `volume` | number | Trading volume during period |
| `market_cap` | number | Market cap at close |

## API Endpoint

```
GET https://api.coinpaprika.com/v1/coins/{coin_id}/ohlcv/historical
  ?start={start}
  &end={end}
  &interval={interval}
  &limit={limit}
  &quote={quote}
```

## Supported Intervals

| Interval | Description | Max History |
|----------|-------------|-------------|
| `5m` | 5 minutes | 1 week |
| `15m` | 15 minutes | 2 weeks |
| `30m` | 30 minutes | 1 month |
| `1h` | 1 hour | 3 months |
| `6h` | 6 hours | 1 year |
| `12h` | 12 hours | 2 years |
| `24h` | 1 day | All history |
